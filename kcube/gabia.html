FTP 접속 및 파일 업로드

[FTP 소프트웨어 사용 매뉴얼]

     
  • FTP 접속은 FTP 소프트웨어를 이용해서 접속할 수 있습니다.
    (※ 호스트 주소: 도메인명 / 계정은 대소문자 구분 필요)
  • 내부 방화벽이나 인터넷 공유기를 사용하는 경우, FTP 접속 시 패시브 모드(Passive mode)에 체크(v) 후 접속해 주세요.
  • 보안을 위해 호스팅 설치 후 1개월 후부터 호스팅 관리에서 FTP 접속 가능 기간을 직접 설정할 수 있습니다.

[홈페이지에 파일 올리기]

  • URL로 접속할 때 가장 먼저 보여줄 파일 이름은 반드시 index로 설정해야 합니다.
  • PHP 사용 시 확장자를 반드시 PHP, PHP3, PHP4 로 설정합니다.
  • 파일 이름에 한글 또는 공백 문자(space)가 있거나, 영문 대소문자가 혼용된 경우 페이지에 연결되지 않을 수 있으니 파일 이름은
    영문 소문자로 통일    하는 것이 좋습니다.

메일 계정 설정

[웹메일 사용 매뉴얼]

       
  • 관리자 모드에서 메일 계정 설정이 완료되어야 메일을 사용할 수 있습니다.
  • 원활한 메일 송수신을 위해 도메인을 ‘통합 White 도메인’으로 반드시 등록해 주세요.
  • 대량 메일 또는 정크메일을 발송할 경우 고객의 동의 없이 메일 서비스를 중단할 수 있습니다.

네임서버

  • 도메인 네임서버가 가비아가 아닌 경우 가비아 네임서버로 변경해야 설치가 됩니다.
  • 타사에서 이전하는 경우 먼저 가비아로 홈페이지를 이전한 후 네임서버를 변경해야 사이트가 중단되지 않습니다.
  • 타사 네임서버를 계속 이용하려면 해당 업체에서 네임서버 정보를 변경해 주세요.
